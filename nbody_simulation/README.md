"# N-body_problem" 

## Как запустить симуляцию

## 
1) Требования
Python 3.8+, numpy, matplotlib, pathlib, sys, importlib, colelctions;
2) Настроить config.py под нужды симуляции, выбрать метод дифференцирования;
3) Выбрать комплектацию системы тел, указав в config.py имя файла;
4) При необходимости создать новую комплектацию тел, указав имена тел, зарегистрированных одновременно в solar_system_data.py и initial_conditions.py;
5) При отсутствии необходимых тел добавить соответствующие в вышеупомянутые файлы согласно формату объектов. указанному в файлах;
6) Запустить n_body.py, после завершения моделирвоаня будет построен график, а результаты будут автоматически сохранены в папку resuts.
